# TankZone.io

2D io-игра про танки с бесконечным процедурным миром и бесконечной прокачкой.
Ванильный JS + HTML5 Canvas, без билда и зависимостей.

## Запуск

Открой `index.html` в браузере. Скрипты грузятся как обычные `<script>`, поэтому работает и с `file://`.

Если браузер блокирует — запусти локальный сервер:

```
python -m http.server 8000
# или: npx serve .
```

и открой `http://localhost:8000/`.

Параметр `?auto=1` пропускает стартовое меню (для быстрого теста).

## Управление

- `WASD` / стрелки — движение
- Мышь — прицел
- ЛКМ / Пробел — огонь
- Shift — форсаж (перегревается)

## Что внутри

- **Бесконечный мир.** Чанки 16×16 тайлов (1024×1024 px), генерация на лету через fBm-шум. 5 биомов: вода (непроходимая), песок, трава, земля, камень. Обстакулы (деревья, кусты, валуны, ящики) расставляются детерминированно. Далёкие чанки выгружаются.
- **Враги.** Тир врагов растёт от дистанции до (0,0). HP, урон, темп стрельбы, дальность — масштабируются. Ящики выбиваются, из них падает XP.
- **Прокачка.** При наборе XP — выбор из 3 карточек: HP, регенерация, скорость, урон, скорострельность, скорость пули, пробитие (rare), мультивыстрел (epic). Без потолка.
- **HUD.** Уровень, XP-бар, счёт, убийства, координаты, HP-бар, мини-карта с биомами и врагами.

## Ассеты

- `assets/tanks/tanks_painted.png` — спрайт-лист танков (Tatermand, OpenGameArt, CC-BY 3.0 — см. https://opengameart.org/content/top-down-painted-tanks).
- `assets/tanks/tanks_a.png` — запасной набор (Tatermand, OGA) на случай если захочется переключить стиль.

Весь остальной рендер (мир, тайлы, пули, взрывы, мини-карта) — процедурный, без внешних ассетов.

## Файлы

- `index.html`, `style.css` — разметка и стили
- `js/utils.js` — математика, хеш, шум
- `js/world.js` — процедурная генерация, чанки
- `js/entities.js` — Player, Enemy, Bullet, XPOrb, Particle
- `js/ui.js` — HUD, меню прокачки, мини-карта
- `js/game.js` — главный цикл, ввод, камера, спавн
